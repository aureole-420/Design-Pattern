Launch : Choose src->view->MainAppFrame.java, on the detailed code of MainAppFrame.java,
		right click, then click Run As -> Java Application, and the whole application will start.

This application is a simple GUI program, which paints various components and shapes on the screen.
The project consists of two parts. In the first part, you can click a button to print a string captured from your input.
In the second part, you can click several buttons at the bottom to paint following shapes: rectangle, triangle, circle, and a composite shape made up with a triangle and rectangle.
